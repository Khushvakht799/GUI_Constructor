# Анализатор ошибок Python

Программа для анализа ошибок Python и создания базы данных паттернов для автоматического рефакторинга.

## Назначение

Анализатор принимает текстовые файлы с ошибками Python (логи, вывод консоли) и:
1. Парсит ошибки, выделяя ключевые элементы
2. Классифицирует ошибки по типам
3. Сохраняет паттерны ошибок в базу данных (JSON)
4. База данных используется программой рефакторинга для автоматического исправления
